Kebijakan Privasi
Amalanku

Terakhir diperbarui: 17 Agustus 2025

Amalanku ("Aplikasi") dirancang untuk membantu umat Muslim menjaga konsistensi ibadah harian. Kebijakan ini menjelaskan data apa yang dikumpulkan dan bagaimana kami menanganinya dengan menghormati privasi Anda.

🔍 Pengumpulan Data

Data yang Dikumpulkan:

  • Data Minimal: Amalanku dirancang untuk mengumpulkan data sesedikit mungkin dan sebagian besar beroperasi secara offline
  • Catatan Amalan: Rekaman amalan harian Anda (shalat, dzikir, membaca Al-Quran) tersimpan hanya di perangkat Anda
  • Preferensi Pengguna: Pengaturan notifikasi, reminder, dan konfigurasi aplikasi (disimpan secara lokal)
  • Waktu Shalat & Masjid Terdekat: Perhitungan waktu shalat dan lokasi masjid terdekat berdasarkan lokasi, diproses secara lokal

Data yang Tidak Dikumpulkan:

  • Informasi Pribadi: Tidak mengumpulkan nama, email, nomor telepon, atau data identitas lainnya
  • Analitik Penggunaan: Tidak memantau cara Anda menggunakan aplikasi
  • Informasi Perangkat: Hanya sesuai kebutuhan fungsi aplikasi
  • Data Akun: Tidak ada akun online atau cloud, semua tetap di perangkat
  • Data Perilaku: Tidak memantau atau menganalisis aktivitas spiritual Anda

Penyimpanan Lokal:

  • Tujuan: Menyimpan catatan amalan, preferensi, dan pengaturan aplikasi secara lokal
  • Lokasi: Semua data tersimpan hanya di perangkat Anda
  • Kontrol: Anda dapat menghapus data kapan saja melalui pengaturan aplikasi

🌐 Izin & Akses Jaringan

Izin yang Diperlukan:

  • Storage: Untuk mengekspor catatan amalan ke PDF sebagai cadangan atau dibagikan
  • Notifikasi: Untuk reminder waktu shalat dan amalan
  • Lokasi (Opsional): Untuk menghitung waktu shalat dan menampilkan masjid terdekat; semua proses menghormati privasi Anda
  • Internet: Diperlukan hanya untuk mengambil konten dari API Amalanku (waktu shalat, doa, dzikir)

Data Pribadi Tidak Dikirim: Aplikasi tidak mengirim info pribadi Anda ke API

Yang Tidak Diakses:

  • ❌ Kamera: Tidak diakses
  • ❌ Kontak: Tidak diakses
  • ❌ Mikrofon: Tidak diakses
  • ❌ Telepon: Tidak diakses

Catatan: Semua pemrosesan data terjadi secara lokal di perangkat Anda. Tidak ada data yang dikirim ke server eksternal atau pihak ketiga.

🛡️ Keamanan & Privasi

  • Pemrosesan Lokal: Semua pelacakan amalan dan perhitungan dilakukan di perangkat
  • Tanpa Transmisi Data: Data amalan tidak pernah meninggalkan perangkat
  • Tidak Ada Server Eksternal: Amalanku tidak menggunakan server atau cloud eksternal
  • Offline: Aplikasi berfungsi sepenuhnya offline
  • Enkripsi Data: Data lokal tersimpan aman dengan enkripsi perangkat

📊 Layanan Pihak Ketiga

Amalanku beroperasi secara mandiri:

  • Tidak Ada Layanan Pihak Ketiga
  • ❌ Tidak Ada Analitik: Tidak menggunakan Google Analytics, Firebase, atau layanan tracking
  • ❌ Tidak Ada Iklan: Aplikasi sepenuhnya bebas iklan
  • ❌ Tidak Ada Cloud: Data tidak tersimpan di cloud
  • ❌ Tidak Ada Integrasi Sosial Media
  • ❌ Tidak Ada Laporan Crash yang Mengumpulkan Data

Berbagi Data:

  • Tidak Ada Berbagi Data: Semua data tetap di perangkat
  • Kontrol Ekspor: Hanya Anda yang dapat mengekspor data sebagai PDF

🍪 Cookies dan Tracking

  • ❌ Tidak Ada Cookies
  • ❌ Tidak Ada Tracking
  • ❌ Tidak Ada ID Iklan
  • ❌ Tidak Ada Fingerprinting

👶 Privasi Anak

Amalanku aman untuk semua usia, termasuk anak-anak. Karena tidak mengumpulkan informasi pribadi dan bekerja offline, tidak ada masalah privasi untuk anak di bawah 13 tahun.

🧑⚖️ Kepatuhan

  • GDPR & CCPA: Karena data tetap lokal, persyaratan minim.
  • Retensi Data: Hanya data lokal di perangkat, bisa dihapus kapan saja
  • Hak Menghapus: Data bisa dihapus sepenuhnya melalui pengaturan atau uninstall

🎯 Kontrol Pengguna

  • Lihat Data: Semua catatan amalan terlihat di aplikasi
  • Ekspor Data: Ekspor ke PDF sesuai keinginan
  • Hapus Data: Bersihkan semua data melalui pengaturan
  • Nonaktifkan Fitur: Matikan notifikasi atau lokasi kapan saja
  • Hapus Aplikasi: Semua data lokal ikut terhapus

🕌 Nilai Islam & Privasi

  • Amanah: Data spiritual Anda dijaga dengan penuh kepercayaan
  • Privasi: Hubungan Anda dengan Allah tetap privat
  • Tidak Menghakimi: Aktivitas spiritual tidak dianalisis
  • Hormat: Menghormati perjalanan spiritual pribadi

📱 Fitur Ekspor PDF

  • Inisiatif Pengguna: Hanya saat Anda memilih
  • Dihasilkan Lokal: PDF dibuat di perangkat
  • Kontrol Anda: Pilih tempat simpan atau bagikan
  • Tidak Ada Transmisi: Kami tidak menerima salinan

  • Pemberitahuan melalui aplikasi
