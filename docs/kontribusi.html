🚀 Langkah Kontribusi

Ada 3 cara untuk berkontribusi, sesuai dengan kenyamanan dan latar belakang Anda:

1
👩‍💻 Untuk Developer / Pengguna GitHub (Rekomendasi)

Fork repository, buat branch baru, lalu submit Pull Request dengan deskripsi yang jelas.

  1. 1 Fork repository ini
  2. 2 Clone ke local machine Anda
  3. 3 Explore folder ./data/ untuk melihat kebutuhan data
  4. 4 Buat branch baru untuk kontribusi Anda
  5. 5 Commit perubahan Anda
  6. 6 Submit Pull Request dengan deskripsi yang jelas 
git clone https://github.com/amalanku/amalanku.github.io
cd amalanku
git checkout -b feature/kontribusi-saya
# Lakukan perubahan
git commit -m "Add: kontribusi saya untuk fitur X"
git push origin feature/kontribusi-saya
🐛 Atau gunakan fitur Issue untuk berdiskusi
2
📂 Cara Lebih Simpel — Google Drive

Cocok untuk kontributor konten, data, atau desain yang tidak terbiasa dengan GitHub.

  1. 1 Buka folder Google Drive kontributor
  2. 2 Baca dokumen README di dalamnya
  3. 3 Upload data atau file Anda di folder CONTRIBUTOR
🔗 Buka Folder Kontributor di Google Drive
